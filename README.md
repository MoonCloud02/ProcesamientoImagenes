## CONTENIDO

Este repositorio contiene materiales de apoyo para el libro "Introducción al Procesamiento Espacial de Imágenes" (2025). Aquí encuentras el libro original, una versión con solo texto en Markdown, y colecciones de respuestas y fuentes para cada capítulo.

Cabe aclarar que hay pequeños errores en las respuestas como que el resultado son 4 pixeles pero se dice que son 3, o que en las tablas de las versiones anteriores había errores de formato que se han corregido en la versión LyX.

**Pueden existir otros errores menores**, por lo que se recomienda revisar cuidadosamente las respuestas.

Archivos y carpetas principales:

- `libro1_2025.pdf` — PDF original del libro.
- `libro1_2025.md` — Conversión a Markdown para lectura o edición rápida. Convertido usando MarkItDown, tiene varios errores de formato y palabras mal escritas que no concuerdan con el PDF original. No usar como referencia principal. Ni para usar con IAs porque puede causar mal contexto.
- `v1/` — Primera versión de las respuestas por capítulo (`CapNRespuestas.md`).
- `v2/` — Versión actualizada de las respuestas por capítulo (recomendado para consultas recientes). Tablas corregidas a mejor formato LaTeX.
- `lyx/` — Archivos fuente en LyX para cada capítulo (útiles para editar o generar versiones alternativas). El cambio fue de tablas se pasaron a formato LyX válido.
- `book_edits/` — Sugerencias de mejoras al libro original, organizadas por capítulo en archivos Markdown.

Cómo usar este repositorio:

- Para leer el libro: usar `libro1_2025.pdf` con un lector de PDF.
- Para navegar el texto en formato de texto plano: abre `libro1_2025.md` en VS Code o cualquier editor Markdown.
- Para revisar respuestas y ejercicios: abre `v2/CapNRespuestas.md` para la versión más reciente, o `v1/` si necesitas la versión inicial. La diferencia son las tablas corregidas en formato LaTeX.

## CONTRIBUCIONES

Las contribuciones son bienvenidas. Cualquier persona que desee mejorar las respuestas o agregar nuevos materiales, por favor abrir un "issue" o enviar un "pull request".

Se puede encontrar mis recomendaciones personales de mejoras al libro en la carpeta `book_edits/` en archivos Markdown por capítulo, por ejemplo `book_edits/Cap1_MejorasLibro.md` para el Capítulo 1.

## RECOMENDACIONES GENERALES

- El manuscrito presenta una falta de consistencia en la forma de referenciar elementos visuales.

Se alterna entre el uso de la abreviatura "Fig." (ej., "Fig.2.3"), la palabra completa "Figura" (ej., "Figura 4.1"), variantes con espaciado inconsistente (ej., "Fig 2.4"), a veces se refiere a una figura sin ningún prefijo (ej., "la 2.5"), otras veces va entre paréntesis, otras veces sin paréntesis, otras veces con doble paréntesis y otras veces entre corchetes.

Esta variabilidad puede confundir al lector y afecta la cohesión del texto. Estandarizar la forma de referenciar figuras mejoraría la claridad y profesionalismo del manuscrito.
